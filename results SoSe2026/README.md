# Vision Assistant Project

## Project Structure

This project is organized into two main directories:

### 1. `source/`

The `source/` directory contains individual models that have been separately tested and are confirmed to be working. This directory serves as a repository of tested and verified machine learning and computer vision models used in the Vision Assistant project.

### 2. `vision-assistant/`

The `vision-assistant/` is the main project directory, which integrates all the individual models from the `sources/` directory to create a comprehensive assistive technology solution.

## Key Components

- **Individual Models**: Located in the `source/` directory, these include various pre-trained models for tasks such as:
  
  - Object Detection
  - Face Recognition
  - Scene Description
  - Speech Recognition
  - Text-to-Speech

- **Main Application**: In the `vision-assistant/` directory, which combines these models into a cohesive assistive technology platform.

## Purpose

This project aims to develop an intelligent vision assistance system that can help individuals with visual impairments by providing real-time descriptions of their environment, detecting objects, recognizing faces, and offering interactive audio feedback.

## Model Repository

Each model in the `source/` directory is:

- Individually tested
- Verified for performance
- Ready for integration into the main Vision Assistant application
